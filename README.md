## 🛠️ Pré-requisitos e Execução Local

Para garantir o funcionamento correto dos testes automatizados, siga os passos abaixo:

### 1. Configuração do Ambiente (Android)
* **Android Studio:** Instale o [Android Studio](https://developer.android.com/studio) e crie um dispositivo virtual (Emulador/AVD).
* **Atualização do Navegador:** No celular virtual, abra a Google Play Store e atualize o **Google Chrome** para a versão mais recente.

### 2. Versão Correta do Chromedriver (Essencial)
O Appium precisa do Chromedriver compatível com a versão do Chrome instalada no emulador.
Que Precisa ser baixada corretamente e posta em uma pasta no dico C da seguinte forma:

C:/Chromedriver/chromedriver.exe

### 3. Execução dos Testes Localmente
Abra o prompt de comando * **(Terminal ADM)**.

Execute o comando * **appium**.

* **Importante:** Mantenha a janela do prompt de comando aberta e o celular virtual.
            Depois apenas dar start na IDE de execução.
            EX: intellij
